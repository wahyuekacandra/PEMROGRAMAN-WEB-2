# PEMROGRAMAN WEB 2

Repository ini berisi latihan PHP untuk mata kuliah Pemrograman Web 2. Setiap subfolder `LAT-PTM-x` berisi file latihan sesuai modul.

## Struktur folder

- `LAT-PTM-1` - latihan1.php
- `LAT-PTM-2` - latihan1.php, latihan2.php, latihan3.php
- `LAT-PTM-3` - latihan1.php, latihan2.php, latihan3.php, latihan4.php
- `LAT-PTM-4` - (kosong)
- `LAT-PTM-5` - (kosong)
- `LAT-PTM-6` - latihan1.php, latihan2.php, latihan3.php

## Keterangan singkat

- `LAT-PTM-1`: Contoh dasar array dan perhitungan jumlah elemen.
- `LAT-PTM-2`: Contoh penggunaan IF dan kontrol alur.
- `LAT-PTM-3`: Contoh penggunaan IF, Switch, dan perulangan dengan PHP.
- `LAT-PTM-4`: Folder belum memiliki file latihan.
- `LAT-PTM-5`: Folder belum memiliki file latihan.
- `LAT-PTM-6`: Contoh array, loop, dan penggunaan form PHP.
